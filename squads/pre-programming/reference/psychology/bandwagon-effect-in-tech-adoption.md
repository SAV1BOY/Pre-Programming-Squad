# Efeito Bandwagon na Adoção de Tecnologia

## Viés/Efeito
**Efeito Bandwagon na Adoção de Tecnologia:** A tendência de adotar tecnologias ou frameworks porque 'todo mundo está usando', sem avaliar se são adequados para o contexto específico do projeto.

## Como se Manifesta em Pré-Programação

- Escolher Kubernetes para um projeto simples porque é popular
- Adotar microserviços sem necessidade real de escala independente
- Usar o framework JS da moda sem avaliar maturidade e suporte
- Migrar para cloud-native sem calcular custo real vs benefício
