# Viés de Autoridade em Decisões Técnicas

## Viés/Efeito
**Viés de Autoridade em Decisões Técnicas:** A tendência de aceitar decisões técnicas sem questionamento quando vêm de figuras de autoridade (tech leads, arquitetos seniores, CTOs) — mesmo quando há evidências contrárias.

## Como se Manifesta em Pré-Programação

- Aceitar arquitetura do principal engineer sem revisão crítica
- Não questionar stack choices do CTO
- Tratar 'best practices' de big techs como verdades absolutas
- Ignorar input de desenvolvedores juniores que têm contexto local
