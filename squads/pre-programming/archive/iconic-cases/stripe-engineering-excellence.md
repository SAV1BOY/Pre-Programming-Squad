# Stripe — Engineering Excellence Case Study

## Contexto
Stripe é reconhecida como uma das empresas com maior rigor de engenharia no mundo, especialmente em design de APIs e documentação técnica.

## Práticas Relevantes para Pré-Programação

### 1. API Design First
- Contratos de API definidos antes de qualquer implementação
- Documentação é cidadã de primeira classe
- Backward compatibility como princípio inegociável

### 2. Design Docs Rigorosos
- Todo projeto significativo requer design doc aprovado
- Template estruturado com: problema, opções, trade-offs, decisão
- Review por multiple stakeholders antes de implementação

### 3. Incremental Rollout
- Feature flags em tudo
- Rollout gradual com monitoramento
- Rollback automático baseado em métricas

## Lições para o Squad
- Design docs não são burocracia — são ferramenta de pensamento
- API-first reduz retrabalho significativamente
- Rollout gradual é seguro de incidentes
