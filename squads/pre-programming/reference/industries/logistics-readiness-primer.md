# Logística e Supply Chain — Readiness Primer

## Contexto da Indústria
Sistemas de logística, gestão de frotas, rastreamento, otimização de rotas e gestão de estoque.

## Desafios Específicos de Pré-Programação

### 1. Integração com múltiplos carriers e APIs de rastreamento

### 2. Real-time tracking com requisitos de latência

### 3. Otimização combinatória (NP-hard) para rotas

### 4. Sazonalidade extrema (Black Friday, Natal)

### 5. Regulamentação de transporte por região

## Checklist de Readiness Específico

### Obrigatório
- [ ] Regulamentação setorial foi mapeada e compreendida
- [ ] Integrações externas foram identificadas com SLAs
- [ ] Requisitos de escala e performance estão quantificados
- [ ] Dados sensíveis estão classificados e protegidos
- [ ] Stack tecnológica é adequada para as demandas do setor

### Recomendado
- [ ] Benchmark de competidores/empresas similares foi realizado
- [ ] Especialista do domínio foi consultado
- [ ] Cenários de pico foram modelados
- [ ] Plano de compliance está documentado
- [ ] Estratégia de go-to-market está alinhada com capacidade técnica
