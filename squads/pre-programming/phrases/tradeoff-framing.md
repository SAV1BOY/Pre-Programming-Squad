# Frases: Enquadramento de Trade-offs

## Contexto
Usar quando é necessário apresentar trade-offs de forma clara para stakeholders ou equipe, evitando a ilusão de que é possível ter tudo sem concessões. Estas frases ajudam a tornar as escolhas explícitas.

## Frases-Modelo

1. "Podemos ter rápido, barato ou completo — mas não os três. Qual é a prioridade?"

2. "Se escolhermos essa abordagem, ganhamos X mas abrimos mão de Y. Estamos confortáveis com isso?"

3. "Essa decisão tem um custo que não aparece agora, mas aparece em 6 meses: manutenção mais complexa."

4. "Ambas as opções são viáveis. A diferença está em: Opção A é mais rápida agora, mas mais cara para manter. Opção B leva mais tempo, mas é mais sustentável."

5. "Não existe solução perfeita. A pergunta é: com qual conjunto de imperfeições preferimos conviver?"

6. "Adicionar essa funcionalidade ao MVP implica em adiar a entrega em X semanas. O valor dela justifica o atraso?"

7. "Simplificar aqui economiza 3 semanas, mas transfere a complexidade para o usuário. É um trade-off aceitável?"

8. "Cada 'sim' a um requisito novo é um 'não' implícito a um prazo ou a outro requisito. Vamos tornar isso explícito."

## Quando Usar
- Quando stakeholders pedem "tudo para ontem".
- Quando a equipe precisa decidir entre opções com prós e contras claros.
- Quando o impacto de uma decisão precisa ser tornado visível.

## Quando NÃO Usar
- Quando não há trade-off real (a decisão é clara).
- Para criar impedimentos artificiais.
- Quando o tom pode soar como "sempre dizemos não".

## Tom Recomendado
Transparente e colaborativo. Trade-offs não são problemas — são ferramentas de decisão. Apresente as opções com clareza e deixe o decisor escolher com informação completa. Nunca use trade-offs como desculpa para não entregar.
