# Efeito IKEA em Decisões de Arquitetura

## Viés/Efeito
**Efeito IKEA em Decisões de Arquitetura:** A tendência de valorizar desproporcionalmente soluções que nós mesmos construímos, resistindo a substituí-las por alternativas melhores.

## Como se Manifesta em Pré-Programação

- Insistir em framework interno quando existem alternativas open-source maduras
- Resistir a refatorar código próprio mesmo quando claramente problemático
- Defender decisões arquiteturais passadas sem reavaliar com novos dados
- NIH (Not Invented Here) syndrome em avaliações de build vs buy
