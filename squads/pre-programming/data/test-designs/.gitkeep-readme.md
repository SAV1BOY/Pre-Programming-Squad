# Test Designs Directory

## Propósito
Armazena designs de teste criados durante a fase de pré-programação, antes de qualquer implementação.

## Conteúdo Esperado
- Estratégias de teste por projeto
- Matrizes de cobertura de requisitos
- Cenários de teste para edge cases
- Critérios de aceitação por feature

## Formato
Cada arquivo segue o template de `templates/test-design-template.md`.
