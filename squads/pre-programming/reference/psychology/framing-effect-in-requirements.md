# Efeito de Enquadramento em Requisitos

## Viés/Efeito
**Efeito de Enquadramento em Requisitos:** A forma como um requisito é apresentado influencia dramaticamente como ele é interpretado e priorizado, independente do conteúdo objetivo.

## Como se Manifesta em Pré-Programação

- Mesmo requisito priorizado diferente dependendo se apresentado como ganho ou prevenção de perda
- Requisitos apresentados por stakeholders seniores recebem prioridade indevida
- Linguagem emocional ('urgente', 'crítico') enviesando priorização
- Dados apresentados de formas diferentes levando a conclusões opostas
